Símbolos e emojis no meu site!

Durante o curso de HTML e CSS, aprendi a utilizar diversos símbolos e emojis para deixar meu site mais interativo e visualmente atraente

Símbolos

Para adicionar símbolos, usamos "&" seguido do código referente ao símbolos que desejamos,
aqui vai uma pequena lista de símbolos aprendidos:

  •   - Espaçço em branco
  • ® - Marca registrada
  • © - Copyright
  • ™ - Trade Mark
  • € - Euro
  • £ - Libra(Pound)
  • ¥ - Yen
  • ¢ - Cent
  • ∅ - Vazio
  • ∑ - Soma
  • Δ - Delta (com "D" maiúsculo, pois com "d" minúsculo é outro símbolo)
  • ← - Seta para a esquerda
  • → - Seta para a direita
  • ↑ - Seta para cima
  • ↓ - Seta para baixo

Emojis

Para adicionar emojis, usamos o "&" seguido do codepoint do emoji desejado, como por exemplo:

  • 😁 - Smiley
  • 😂 - Chorando de Rir
  • 😃 - Rindo Forte
  • 😄 - Rindo com Olhos Felizes
  • 😅 - Rindo com Suor

Não sabe o codepoint de um emoji? Acesse o site Emojipedia para encontrar os códigos!
